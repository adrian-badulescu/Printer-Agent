URS Printer Agent — pachet MSI (WiX), nu MSIX

Acest proiect produce un installer MSI care înregistrează serviciul Windows URSPrinterAgent
(pornire automată, LocalSystem). Nu este un pachet MSIX / App Installer.

Pentru distribuire tip MSIX, ar fi necesar un proiect separat (Package.appxmanifest,
App Installer) sau un pipeline de conversie MSI→MSIX.
